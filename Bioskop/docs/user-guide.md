# Panduan Pengguna Bioskop

## 🎫 Cara Memesan Tiket

1. **Login** ke akun Anda (atau register jika belum punya)
2. Buka halaman **Pesan Tiket** dari sidebar
3. **Step 1**: Pilih film yang ingin ditonton, lalu pilih jadwal tayang
4. **Step 2**: Pilih kursi di seat map (klik kursi yang tersedia — warna hijau)
   - Hijau = Tersedia | Ungu = Premium | Oranye = VIP | Abu = Terisi
5. **Step 3**: Tambah snack/minuman (opsional), pilih metode pembayaran
6. Klik **Bayar** untuk menyelesaikan pesanan
7. Tiket QR Code akan muncul — bisa dilihat di **Tiket Saya**

## 💬 Curhat Film

- Timeline seperti Twitter untuk komunitas film
- Posting teks, upload gambar/video/dokumen
- Gunakan `#hashtag` untuk topik trending
- Like & komentar postingan orang lain
- Posting event untuk nobar bareng

## 🤖 Chat dengan Si Bobby

- Tanya rekomendasi film, jadwal, snack, atau trivia film
- Upload gambar untuk dianalisis
- Multi-session: bisa punya beberapa chat paralel
- Reset session untuk memulai percakapan baru

## 👤 Profil

- Edit nama, email, password
- Upload foto profil (klik icon kamera di avatar)
- Semua perubahan disimpan otomatis
