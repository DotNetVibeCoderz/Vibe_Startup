# 📖 USER_GUIDE.md - Panduan Penggunaan LandLord

## 🏠 Halaman Utama (Home)
Menampilkan overview aplikasi, quick stats, dan navigasi ke fitur utama.

---

## 🗺️ Maps Page
1. **Browsing Properti**: Gunakan peta interaktif di sisi kiri
2. **Search**: Ketik keyword di kolom pencarian untuk mencari tanah/bangunan
3. **Filter Tipe**: Pilih "Semua", "Tanah", atau "Bangunan"
4. **Detail Panel**: Klik properti di daftar kanan untuk melihat metadata lengkap
5. **Mode 2D/3D**: Gunakan tombol toggle untuk beralih mode peta (perlu API key)

---

## 📊 Master Data Page
### Data Tanah
1. Pilih tab "Data Tanah"
2. Gunakan filter Jenis Hak dan Status Pajak
3. Klik "Tambah Tanah" untuk menambah data baru
4. Isi form dengan metadata standar Indonesia
5. Klik ✏️ untuk edit, 🗑️ untuk hapus

### Data Bangunan
1. Pilih tab "Data Bangunan"
2. Gunakan filter Jenis Bangunan
3. Operasi CRUD sama seperti data tanah

### Export
- Klik "Export CSV" untuk mendownload data dalam format CSV

---

## 📈 Dashboard Page
- **Summary Cards**: Total tanah, luas, bangunan, status pajak
- **Charts Distribusi**: Bar chart interaktif jenis hak dan bangunan
- **AI Insights**: Rekomendasi otomatis dari data
- **Recent Data**: Tabel data terbaru

---

## 🤖 Chat - Frengky Ganteng
1. **New Chat**: Klik "New Chat" untuk sesi baru
2. **Multi-Session**: Pilih sesi di panel kiri
3. **Kirim Pesan**: Ketik pertanyaan dan tekan Enter/Kirim
4. **Lampiran**: Klik 📎 untuk melampirkan file
5. **Reset**: Klik 🔄 untuk mereset sesi
6. **Hapus**: Klik 🗑️ untuk menghapus sesi

### Contoh Pertanyaan:
- "tampilkan data tanah"
- "berapa total bangunan?"
- "statistik tanah"
- "help"

---

## ⚙️ Settings Page (Admin Only)
- **LLM Provider**: Pilih OpenAI, Anthropic, Gemini, atau Ollama
- **Vector DB**: Pilih provider vector database
- **Storage**: Pilih tempat penyimpanan file
- **Database**: Pilih provider database
- **Chat Bot**: Atur persona dan perilaku bot
