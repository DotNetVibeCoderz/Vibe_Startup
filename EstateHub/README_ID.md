# 🏠 EstateHub - Platform Manajemen Properti

[![.NET](https://img.shields.io/badge/.NET-10.0-blueviolet)](https://dotnet.microsoft.com/)
[![Blazor](https://img.shields.io/badge/Blazor-Server-purple)](https://dotnet.microsoft.com/apps/aspnet/web-apps/blazor)

## ✨ Fitur Unggulan

### 🏠 Pembeli/Penyewa
- Pencarian properti dengan filter lengkap
- Peta interaktif dengan pin properti
- Simulasi KPR & kalkulator cicilan
- Chat dengan agen via aplikasi & WhatsApp
- Virtual tour & video properti

### 🏢 Pemilik/Agen
- Listing & manajemen properti
- Dashboard penjualan & performa
- Kontrak digital dengan e-signature
- CRM leads & follow-up
- Iklan & promosi premium

### ⚙️ Admin
- Manajemen user & verifikasi
- Monitoring transaksi
- Laporan & analitik lengkap

### 🤖 AI "Tante Rita"
- Chatbot 24/7 untuk tanya properti
- Rekomendasi properti personal
- Bantuan KPR & pajak

---

## 🚀 Quick Start

```bash
# Pastikan .NET 10 terinstall
dotnet --version

# Clone & run
git clone https://github.com/gravicode/estatehub.git
cd EstateHub
dotnet run

# Buka browser di https://localhost:5001
```
