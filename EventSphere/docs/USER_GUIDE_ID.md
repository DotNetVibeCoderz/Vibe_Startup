# 📖 Panduan Pengguna EventSphere (ID)

## Selamat Datang di EventSphere!

EventSphere adalah platform all-in-one untuk merencanakan, mengelola, dan mengeksekusi event atau wedding Anda.

---

## 🔐 Memulai

### Registrasi
1. Buka `/register`
2. Isi nama lengkap, email, password
3. Pilih role: **Client** (perencana event), **Vendor** (penyedia jasa), atau **Guest** (tamu undangan)
4. Setujui syarat & ketentuan, klik **Daftar Sekarang**
5. Login dengan akun baru

### Login
1. Buka `/login`
2. Masukkan email & password
3. Centang "Remember me" untuk sesi 7 hari
4. Klik **Sign In**

## 📅 Dashboard

Setelah login, Anda akan melihat **Dashboard** dengan:
- **Statistik Cards**: Total Events, Upcoming, Vendors, Revenue, Tasks, Rating
- **Quick Actions**: Events, Vendor Directory, Tante Sherly AI
- **Monthly Overview**: Tabel ringkasan bulanan

---

## 📅 Events

### Membuat Event Baru
1. Klik **➕ New Event** di Dashboard atau sidebar
2. Isi form: Nama, Tanggal, Lokasi, Tema, Tipe, Budget, Tamu
3. Klik **🎉 Create Event**

### Mengelola Event
- **List**: Lihat semua event, filter by status, search
- **Detail**: Klik event → lihat ringkasan + quick links ke Guests, Budget, Tasks, Chat, Gallery, Documents, Seating

---

## 👥 Guest Management

1. Pilih event dari grid → masuk ke halaman Guests
2. **➕ Invite Guest**: Masukkan email tamu, pilih role (Guest/VIP/Family)
3. **RSVP Tracking**: Update status tamu lewat dropdown (Pending/Accepted/Declined)
4. **Seat Assignment**: Klik 🪑 → pilih meja & nomor kursi
5. **Plus One**: Centang "Plus One?" saat mengundang

---

## 💰 Budget Tracking

1. Pilih event → lihat ringkasan budget (Total / Terpakai / Sisa)
2. **➕ Add Item**: Tambah item dengan kategori, estimasi, aktual, status bayar
3. Progress bar menunjukkan persentase pemakaian budget

---

## ✅ Tasks & Checklist

1. **My Tasks**: Lihat semua task yang ditugaskan ke Anda
2. **Per Event**: Kelola task spesifik untuk event
3. Priority: Low 🟢 → Medium 🟡 → High 🟠 → Urgent 🔴
4. Klik **✅ Done** untuk menandai selesai

---

## 🏢 Vendor Directory

1. Browse vendor berdasarkan kategori (Catering, Dekorasi, Fotografi, dll)
2. Lihat rating, review, price range
3. Admin bisa **Add/Edit/Delete** vendor
4. Search vendor by nama atau kategori

---

## 💬 Chat & Messaging

1. Real-time chat per event
2. Buat chat session baru
3. Kirim pesan teks

---

## 🤖 Tante Sherly AI

ChatBot AI yang bisa:
- Menjawab pertanyaan event planning
- Mencari vendor di database
- Mengecek budget dan task
- Mencari informasi di internet (Tavily)
- Membaca halaman web (scraping)
- Menghitung budget estimasi
- Konversi mata uang & satuan
- Tips & checklist event

### Multi-Model Support
Pilih model AI dari dropdown: OpenAI, Anthropic, Gemini, Ollama

### Multi-Session
- **➕ New Chat**: Buat percakapan baru
- **🔄 Reset**: Hapus history chat
- **✕**: Hapus session

### Upload Gambar
Klik 📎 → pilih gambar → AI akan "melihat" dan menganalisis gambar

---

## 🖼️ Gallery

1. Pilih event → upload foto & video
2. **Klik gambar** untuk preview ukuran besar
3. Filter by category (Ceremony, Reception, Candid)

---

## 📁 Documents

1. Upload kontrak, proposal, invoice (.pdf, .doc, .xlsx)
2. Download file yang sudah di-upload
3. Export laporan event ke Excel (multi-sheet)

---

## 🪑 Seating Planner

1. Visual canvas untuk tata letak meja
2. **➕ Add Table**: Tambah meja (Round/Rectangle/Square)
3. Klik meja → edit nama, kapasitas, warna
4. Lihat tamu yang duduk di setiap meja

---

## 💡 Forum

1. Buat post baru (Tips/Review/Discussion/Question)
2. Klik post → lihat detail + komentar
3. **❤️ Like** post (sekali per user)
4. **⭐ Rating** 1-5 bintang
5. **💬 Komentar**: Tulis & hapus komentar
6. Author bisa edit/delete post sendiri

---

## ⚙️ Admin Panel

(Admin Only) `/admin/users`
- **User Management**: CRUD, filter, sort, paging
- **Role Management**: Assign/unassign multiple roles
- **Toggle Active/Inactive**: Klik badge status
- **Export**: CSV & Excel download

---

## 👤 Profile

1. Edit nama, perusahaan, bio, timezone
2. Upload avatar
3. Ganti password
