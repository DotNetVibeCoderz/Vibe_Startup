# 🎉 EventSphere - Platform Event & Wedding Organizer

> **Solusi Perencanaan Event All-in-One**  
> Dibangun dengan Blazor Server .NET 10 • Bertenaga AI • UI Neomorphism Modern

---

## 🚀 Mulai Cepat

### Prasyarat
- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- SQLite (default) atau SQL Server / MySQL / PostgreSQL

### Menjalankan
```bash
cd EventSphere
dotnet restore
dotnet run
```

Buka `https://localhost:5001` di browser.

## ✨ Fitur

### 🔑 Fitur Inti
- ✅ Pembuatan & Manajemen Event (CRUD, tracking status)
- ✅ Manajemen Tamu (RSVP, seating arrangement, dietary)
- ✅ Direktori Vendor (database, kontrak, invoice)
- ✅ Tracking Budget (estimasi, alokasi, monitoring)
- ✅ Task & Checklist (deadline, pengingat, progress)

### 💬 Komunikasi
- ✅ Chat & Messaging (real-time, per-event)
- ✅ Berbagi Dokumen (kontrak, proposal, foto)
- ✅ Sistem Notifikasi (deadline, pengingat)

### 🎨 Desain
- ✅ Pemilih Tema & Gaya
- ✅ Visual Seating Planner
- ✅ Galeri Foto & Media

### 🤖 AI (Tante Sherly)
- ✅ ChatBot Multi-model (OpenAI, Anthropic, Gemini, Ollama)
- ✅ Integrasi Semantic Kernel
- ✅ Pencarian Internet (Tavily API)
- ✅ Web Scraping
- ✅ Rendering Markdown
- ✅ Upload Gambar
- ✅ Multi-session

### 📊 Analitik
- ✅ Dashboard dengan Metrik Utama
- ✅ Statistik Bulanan
- ✅ Export CSV & Excel

## 👥 Matriks Peran

| Peran | Akses |
|-------|-------|
| **Admin** | Akses penuh, kelola user & vendor, monitoring budget |
| **Organizer** | Kelola event, checklist, vendor, tamu, komunikasi |

## 🔧 Konfigurasi

Lihat `appsettings.json` untuk konfigurasi:
- **Database**: SQLite / SqlServer / MySQL / PostgreSQL
- **Storage**: FileSystem / AzureBlob / S3 / MinIO
- **AI**: OpenAI / Anthropic / Gemini / Ollama
- **ChatBot**: System prompt, temperature, model
